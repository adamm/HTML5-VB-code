Reference

<audio> Tag

Synopsis

<audio controls autoplay preload='type'>
  <source src="music.mp3" type='audo/mpeg'>
  <source src="music.ogg" type='audio/ogg; codecs="vorbis"'>
  <source src="music.wav" type='audio/x-wav'>
  <!-- Fallback Content -->
</audio>

Description

Note the type attribute can be omitted, or it can be only the file's MIME type or the MIME plus its codec. The latter is recommended, when available.
