Reference

contenteditable Attribute

Synopsis

<p contenteditable='true'>Editable Paragraph Text</p>

Description

You can configure any static text block to become editable by the user within any HTML5 web browser using the new contenteditable attribute. When activated, the text block will appear normal, but once clicked will transform into a form input field pre-populated with the current text.

Example

Let four captains Bear Hamlet, like a soldier, to the stage; for he was likely, had he been put on, to have proved most royally: and, for his passage, the soldiers' music and the rites of war speak loudly for him. Take up the bodies: such a sight as this becomes the field, but here shows much amiss. Go, bid the soldiers shoot.
