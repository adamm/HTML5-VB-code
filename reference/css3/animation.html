Animation Properties

Synopsis

<style>
  selector {
    -webkit-animation-name: name;
    -webkit-animation-direction: direction;
    -webkit-animation-duration: duration;
    -webkit-animation-iteration-count: count;
    property: value1;
    ...
  }
  @-webkit-keyframes name {
    percent1 { property: value2; ... }
    percent2 { property: value3; ... }
    ...
  }
</style>

Description

One of the highly anticipated features of CSS3 is its ability to create simple animations without any JavaScript coding. Implemented by Apple in the WebKit engine, a new group of animation properties and a new selector called @-webkit-keyframes were created which expands on the idea of transitions-between-styles, described in the previous section. To begin, compose a selector that matches what you would like to animate.
