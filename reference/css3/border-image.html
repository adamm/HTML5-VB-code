Border Image Property

Synopsis

<style>
  selector {
    border-image: source slice repeat;
    -o-border-image: source slice repeat;
    -moz-border-image: source slice repeat;
    -webkit-border-image: source slice repeat;
    border-width: width;
  }
</style>

Description

You can create custom borders around elements with the help of image templates and CSS3â€™s new border-image property. Web browsers that support this feature will dynamically map your template image to correctly stretch and shrink its edges to match the selected object, but leave the corners intact.

Example

Note that Opera, Mozilla, and Webkit browsers have their own property names, but accept the same value list. As a result, you need to define your border-image definition three more times.

It is also a good idea to also define the regular border-style and border-color properties, just incase you encounter a browser that does not understand border-image, such is the case with all versions of Internet Explorer 9 and earlier.
