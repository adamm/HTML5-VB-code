Border Radius Property

Synopsis

<style>
  selector {
    border: width style color;
    border-radius: radius;
    -moz-border-radius: radius;
    -webkit-browser-radius: radius;
  }
</style>

Description

It was not that long ago when HTML developers had to use images, tables, and patience just to produce seamless rounded borders. Thanks to CSS3, you can now modify any element that supports the border property, and render rounded borders seamlessly.

The border radius feature extends upon the border-width, border-style and border-color properties, or the border shorthand property, specified by CSS1, requiring you to specify them too.

Example

Since this feature has only recently been standardized in CSS3, the actual property name varies depending on the web browser engine version. Older WebKit browsers like Chrome and Safari use the -webkit- prefix, older Mozilla browsers like Firefox use the -moz- prefix. Newer versions of these browsers, plus Opera and Internet Explorer 9, use no special prefix.
