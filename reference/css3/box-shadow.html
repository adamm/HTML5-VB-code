Box Shadow Property

Synopsis

<style>
  selector {
    box-shadow: xpos ypos blur spread color, ...;
  }
</style>

Description

You can easily add a shadow effect to any element with the box-shadow property in CSS3. This effect requires at least two values, the relative X and Y positions, but accepts up to six values to fine-tune the shadow application.

Example

Most HTML5 web browsers do honor the box-shadow property name, but this is very recent. Browsers like Chrome 9 and earlier, Safari 3.0 and Firefox 3.5 instead use their own derivative property name.

Therefore, you should also specify -moz-box-shadow and -webkit-box-shadow to support these browsers. The non-standard property names do accept the same prototype as the standard box-shadow.
