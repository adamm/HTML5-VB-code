Flexible Box Model

Synopsis

<style>
  div.flex-box {
    display: box;
    display: -moz-box;
    display: -webkit-box;
  }
</style>
<div class='flex-box'>
  <p>Child 1</p>
  <p>Child 2</p>
  <p>Child 3</p>
</div>

Description

The Flexible Box Model is a module of the draft CSS3 specification, and by extension, HTML5. It allows you to organize web page data into multiple rows or columns within a new display: box object.

The Flexible Box Model makes it is easier to construct one-dimensional data -- one row with multiple columns or one column with multiple rows -- than HTML tables, and provides additional flexibility when ordering and stylizing the data in your CSS rules.
