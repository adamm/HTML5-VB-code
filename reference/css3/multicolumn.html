Multi-column Layout

Synopsis

<style>
  div.multi-column {
    column-count: number
    -moz-column-count: number
    -webkit-column-count: number
  }
</style>
<div class='multi-column'>
  <p>paragraph 1</p>
  <p>paragraph 2</p>
  <p>paragraph 3</p>
</div>

Description

The Multi-column Layout is a module of the draft CSS3 specification, and by extension, HTML5. It allows you to organize web page data into multiple columns of variable height and width, depending on how much content there is to display.

The Multi-column Layout makes it is easier to read large amounts of text, and allows for better use of space, just like columns in a newspaper. This newspaper-column style format has been used in print for centuries, and now the layout style can easily be transferred to web pages with very little code change.
