Scrollbar Pseudo-selector

Synopsis

<style>
  selector {
    overflow: auto;
    display: block;
    width: windowwidth;
    height: windowheight;
  }
  ::webkit-scrollbar {
    width: scrollbarwidth;
    height: scrollbarheight;
  }
  ::webkit-scrollbar-button {
  }
  ::webkit-scrollbar-thumb {
  }
  ::webkit-scrollbar-track {
  }
  ::webkit-scrollbar-track-piece {
  }
</style>

Description

You can create a custom scrollbar on any display block that has the overflow property set. This can be used to replace the bland scrollbar that is built-in to the browserâ€™s operating system. This feature is supported in only in desktop WebKit browsers, like Chrome and Safari browsers, but is not available in FireFox 4, Internet Explorer 9, or Opera 11. This is because WebKit introduced and implemented the feature, and it is not yet included by the W3C for CSS3.
