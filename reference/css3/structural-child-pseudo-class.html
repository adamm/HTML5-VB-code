Structural Child Pseudo-class Selectors

Synopsis

<style>
  element:first-child { ... }
  element:last-child { ... }
  element:only-child { ... }
  element:nth-child(counter) { ... }
  element:nth-last-child(counter) { ... }
</style>

Description

The pseudo-class is a list of pre-defined names that the web browser users to match a specific structure of elements. In CSS2, the first child pseudo-class was introduced, :first-child, to match the first child element of its parent but nothing more. In CSS3, the child list was expanded to allow you to match new combinations and patterns of child elements.
