Structural Type Pseudo-class Selectors

Synopsis

<style>
  element:first-of-type { ... }
  element:last-of-type { ... }
  element:only-of-type { ... }
  element:nth-of-type(counter) { ... }
  element:nth-last-of-type(counter) { ... }
</style>

Description

The pseudo-class is a list of pre-defined names that the web browser users to match a specific structure of elements. The most common pseudo-class is :hover, which is used to alter an elementâ€™s style whenever the mouse hovers overtop. In CSS3, the list of pseudo-class names was expanded in two new areas: type and child. The type pseudo-class is described here, the child pseudo-class is on the next page.
