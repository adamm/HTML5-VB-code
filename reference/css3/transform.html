CSS Reference

Transform Property

Synopsis

<style>
  selector {
    transform: scale( xvalue, yvalue );
    transform: skew( xdegrees, ydegrees );
    transform: rotate( degrees );
    transform: translate( xpos, ypos );
  }
</style>

Description

You can transform any type of block object in CSS3 using the new transform property. This allows you to scale, skew, rotate, and translate objects by re-rendering them in very interesting ways in supporting web browsers. This feature will become useful later when you start animating with CSS3.
