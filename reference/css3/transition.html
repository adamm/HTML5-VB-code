Transition Property

Synopsis

<style>
  selector {
    transition: property, duration, timing, delay;
    property: value1;
  }
  selector:hover {
    property: value2;
  }
</style>

Description

Transitions are a very easy way to create simple morphing animations of an object as it changes from one style into another, using nothing but CSS rules. To do this, you must define a starting and ending selector that the target object will be matched with. For mouse-based animations, often the two selector will identify the same HTML object, except the second will simply append the :hover pseudo-class.
