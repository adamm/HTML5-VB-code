<figcaption> Tag

Synopsis

<figure>
    <img src='image.jpg'>
    <figcaption>Caption Text</figcaption>
</figure>

Description

Place the <figcaption> tag either above or below the image, table, or code within the <figure> group. Within this tag, insert your caption text.

Most HTML5 web browsers will not format the figure or figcaption elements using a pre-defined CSS template. This means that they will appear as nothing special, just text following an image, when viewed by the browser. You should use figure and figcaption CSS selectors to add in some display properties into your stylesheet, like the font, border, margin, and padding.
