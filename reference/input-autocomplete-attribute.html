Input Auto-complete Attribute

Synopsis

<input type='text' name='field' value='default' autocomplete='off'>

Description

You can use the autocomplete input attribute to instruct the browser how it should remember the values in a form after it is submitted. If enabled, when the form loads again, supported web browsers will present a pull-down list showing all previous values used for this field. The actual values are displayed after the user begins to type the first character that matches a previously used value. This is a useful feature to provide for commonly-entered fields, like a first and last name, or a mailing address, but should be disabled for security fields, like usernames and Q&A authentication..
