Input Datalist Pull-down

Synopsis

<input type='text' name='field' value='default' list='listid'>
<datalist id='listid'>
  <option label='label' value='value'></option>
  ...
</datalist>

Description

If you have a text input field that supports text input, and you want to provide a subtle list of suggested values, you create a drop-down list of optional suggestions that will appear immediately under the form input as soon as the user starts typing. The partial text input actually acts as a filter in this drop-down list, only options matching the filter will be listed.
