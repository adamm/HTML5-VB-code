Input Date Type

Synopsis

<input type='date' name='field' value='yyyy-mm-dd'>

Description

You can use the date input type, new in HTML5, to restrict input fields to values formatted like a date or time string. Supported web browsers will reject values that do not conform to the yyyy-mm-dd format.
