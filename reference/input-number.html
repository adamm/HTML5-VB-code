Input Number Type

Synopsis

<input type='number' name='field' value='number'
    min='number' max='number' step='number'>

Description

You can use the number input type to restrict input fields to numeric values only. Supported web browsers will inhibit any non-numeric data from being submitted through this input type by purging any such data as soon as it is typed by the user.

Additionally, the number input type also allows you to set a minimum and maximum number range, as well as an interval. If a number is outside of the specified range, or is not divisible by the stepping value, it will be rejected with an automated popup message.
