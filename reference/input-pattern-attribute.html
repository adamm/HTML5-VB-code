Input Pattern Attribute

Synopsis

<input type='text' name='field' value='default' pattern='regex' title='text'>

Description

You can use the pattern matching attribute, new in HTML5, to restrict input fields to values that follow a custom formatting pattern you define. This is useful if you need to ensure that the user submits correctly formatted values, otherwise display a message to the user and stop the form submission process.
