Input Range Type

Synopsis

<input type='range' name='field' value='number' 
    min='number' max='number' step='number'>

Description

You can use the range input type, new in HTML5, to create a sliding input field. Supported web browsers will display a slider bar and a thumb-piece that can be moved left and right. The value of the input element changes depending on the position of the thumb piece on the slider bar.
