Input Telephone Type

Synopsis

<input type='tel' name='field' value='number'>

Description

You can use the telephone input type to accept into an input field telephone numbers and characters like +, -, and space. When a web form requires a phone number for an input field, it is relatively simple for desktop users to simply type in a phone number, but on mobile web browsers there is no point in displaying a full-sized virtual keyboard, instead only a numeric keypad with the telephone-related characters appears.
