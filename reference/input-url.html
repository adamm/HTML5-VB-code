Input URL Type

Synopsis

<input type='url' name='field' value='url'>

Description

You can use the URL input type, new in HTML5, to restrict input fields to values formatted like an absolute HTTP or HTTPS Internet address. Supported web browsers will reject values that are not formatted with an “@”-sign and a domain name with a TLD, like user@domain.com.

Some web browsers will even attempt to sanitize the URL, automatically removing any line breaks and white-space from the value before it is submitted.
