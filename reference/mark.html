<mark> Tag

Synopsis

<mark>text to highlight</mark>

Description

You can highlight text in a web page, just like a highlighter pen on paper, using the new mark element in HTML5. This allows you to put additional a new emphasis on a block of text, without using the older methods like italicizing, bolding, or underlining.
