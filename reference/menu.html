Reference

<menu> Tag

Synopsis

<menu type='toolbar'>
    <li><menu type='context' label='Title'>
        <li>Option</li>
        ...
    </menu></li>
    ...
</menu>

Description

Desktop-style pull-down menus have been featured on many web sites for quite some time, but many implementations are complicated to code and sloppy to use. New in HTML5 is the menu element which simplifies the code required to create this feature.
