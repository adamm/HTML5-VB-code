Reference

<meter> Tag

Synopsis

<meter value='number'
       min='number'
       max='number'
       low='number'
       high='number'
       optimum='number'>
    Fallback Display
</meter>

Description

The meter element, new for HTML5, allows you to display a gauge graphic to the user at any point in a web page. This is useful to display some a metered bar to users absolute values.

The meter element accepts several attribute values as input. At minimum, you should set a "value" attribute as a real number between 0 and 1. If you also specify "min" and "max" numbers, then the "value" attribute must be anywhere between "min" and "max" and the browser calculates the display as a percentage meter.

Extra numeric attributes that you can set include "low", "high", and "optimum" which alter the meter's color depending on which are used. For example, use all three extra attributes and the meter's value is between 0 and "low" is colored red; between "low" and "high", yellow; and between "high" and "optimum", green. If "optimum" is omitted, values below "low" and above "high" is yellow, between "low" and "high" is green. Experiment with these attributes to find the color combination that is appropriate for your meter.
