<progress> Tag

Synopsis

<progress value='number' max='number'>
    Fallback Display
</progress>

Description

A progress element differs from the meter element in that its design specification says it moves to convey an in-progress task. This could be something like downloading a file, or processing a command, basically an action that has a relatively short time until completed, but is not an instantaneous. In order to make use of this, some additional JavaScript code is required to control the progress meter's current level.
