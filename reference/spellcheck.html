spellcheck Attribute

Synopsis

<form>
    <textarea spellcheck='true'>Spelchek textarea input</textarea>
</form>

Description

Most web browsers provide a useful built-in spell-check feature available to any web site that allows the user to type text. This feature is enabled when a spellcheck attribute is enabled on an editable text block. Note that content displayed on a site that is not editable is not spell-checked.

This technique can apply to almost any editable text block. This is possible because spellcheck is a new global attribute in HTML5. It is most common to see this on form text or textarea inputs.
