<video> Tag

Synopsis

<video height='size' width='size' controls autoplay preload='type'>
  <source src="movie.mp4" type='video/mp4; codecs="avc1.640029, mp4a.40.2"'>
  <source src="movie.webm" type='video/webm; codecs="vp8, vorbis"'>
  <source src="movie.ogv" type='video/ogg; codecs="theora, vorbis"'>
  <!-- Fallback Content -->
</video>

Description

Note the type attribute can be omitted, or it can be only the file's MIME type or the MIME plus its codec. The latter is recommended.
