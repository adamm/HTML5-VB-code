<wbr> Tag

Synopsis

ThisIsA<wbr>LongWord<wbr>ThatCanBe<wbr>BrokenUp

Description

The wbr element allows you to specify a word-break opportunity allowing long words to span multiple lines. This has a slightly different execution from the soft-hyphen character, introduced in HTML 4.0. While the <wbr> tag will just break up a long word, the &shy; code will also insert a hyphen before the line break.

Example

"And a special treat from the 23rd century, Miss Ria Paschelle, as you all know, was the inventor of the statiophonicoxygeneticamplifiergraphiphonideliverberator. Kind of hard to imagine the world before we had them, isn't it?"
